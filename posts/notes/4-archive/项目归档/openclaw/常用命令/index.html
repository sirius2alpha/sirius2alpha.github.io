OpenClaw 常用命令

April 10, 2026 · 1 min · 29 words · sirius1y

查看容器内环境变量

docker compose exec openclaw-gateway printenv

重启容器

docker compose down && docker compose up -d

查看日志

docker logs -f openclaw-openclaw-gateway-1

docker logs --tail 30 openclaw-openclaw-gateway-1 | grep -E "(telegram|orphaned|stall)"
