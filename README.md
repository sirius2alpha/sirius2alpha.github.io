## 📔 mono for me

-   要使用blog-commit.sh脚本，要在根目录下新建一个 config.sh 文件，内容如下：

    ```shell
    # config.sh
    BLOG_HOME=
    ```

-   项目的git仓库其中含有两个submodule，分别是`Notes`和`paperMod`，在执行完`git clone`后，需要执行`git submodule init`和`git submodule update`来初始化submodule。
